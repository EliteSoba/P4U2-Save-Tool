P4U2 Save Tool

A simple tool where you can upload your P4U2 Steam save data and it'll return a copy with a lot unlocked
Save data is generally located in AppData/Roaming/P4U2/Save
Warning: I haven't tested this across a bunch of accounts and saves (because I only have one account) so I can't guarantee that this works for everyone.
You should back up your save.dat file before trying this tool.
